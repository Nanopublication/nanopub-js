# nanopub-js

A TypeScript library for creating, signing, publishing, and querying nanopublications.

**Status:** This repository is a work in progress. No public release is available yet. A first release will be published once the core functionality is complete.

The initial release will include:

* Creation of nanopublications
* Signature generation and verification
* Publishing to nanopublication servers
* Query utilities
